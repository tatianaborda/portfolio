Portafolio (Proyectos Destacados)

App para consultoría empresarial

React

Fecha: 11/2019

Podés verlo en: https://github.com/tatianaborda/equilatera_front

Este proyecto es el producto de un mes de arduo trabajo en conjunto con personas maravillosas que tuve el gusto de conocer en la Hackatón WeGirlsHack organizada por MeT Cba(Mujeres en tecnología-Córdoba). Me encargué de hacer las pantallas de preguntas para un formulario que resultó en una aplicación-cuestionario para medir las políticas de equidad y género en empresas. Utilicé HTML, CSS y REACT como herramientas para el desarrollo.
